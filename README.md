# Diwali_Sales_Analysis

 Diwali Sales Data Analysis (Python EDA Project)
 Project Overview
This project performs Exploratory Data Analysis (EDA) on a Diwali Sales dataset using Python.
The goal of this analysis is to understand customer purchasing behavior and identify key insights such as:
•	Which gender spends more?
•	Which age group contributes most to sales?
•	Which states generate the highest revenue?
•	Which occupations and product categories perform best?
•	Who are the most valuable customers?

 Tools & Technologies Used
•	Python
•	Pandas – Data Cleaning & Manipulation
•	Matplotlib – Data Visualization
•	Seaborn – Statistical Data Visualization

 Dataset Information
The dataset contains sales transaction data during the Diwali festival, including:
•	Gender
•	Age & Age Group
•	State
•	Marital Status
•	Occupation
•	Product Category
•	Orders
•	Amount

 Data Cleaning Steps
1.	Removed unnecessary columns (Status, unnamed1)
2.	Checked for null values
3.	Dropped rows with missing values
4.	Converted Amount column from float to integer
5.	Generated summary statistics for numerical columns

 Exploratory Data Analysis (EDA)
The following analyses were performed:
 Gender Analysis
•	Count of male vs female customers
•	Total sales amount by gender
 Insight: Female customers contribute more to overall sales.

 Age Group Analysis
•	Orders by age group
•	Total sales by age group
 Insight: Age group 26–35 years contributes the highest revenue.

 State Analysis
•	Top 10 states by number of orders
•	Top 10 states by total sales
 Insight: Uttar Pradesh, Maharashtra, and Karnataka generate the highest sales.

 Marital Status Analysis
•	Purchase distribution by marital status
•	Sales comparison by gender and marital status
 Insight: Married women are major contributors to total sales.

 Occupation Analysis
•	Orders by occupation
•	Sales by occupation
 Insight: Customers working in IT, Healthcare, and Aviation spend more.

Product Category Analysis
•	Most purchased product categories
•	Top 10 most sold products
 Insight: Food, Clothing, and Electronics categories perform the best.

Key Business Insights
•	Married women aged 26–35 years
•	From Uttar Pradesh, Maharashtra, and Karnataka
•	Working in IT, Healthcare, and Aviation
 Are more likely to purchase products from Food, Clothing, and Electronics categories.
